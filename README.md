# Bitwise-algorithms
Binary arithmetic and bit manipulation programs including two’s complement conversion, 16-bit addition, and rightmost bit reset implemented using low-level programming techniques.

## Binary Bitwise Algorithms

This project contains a collection of low-level programs that demonstrate binary arithmetic and bit manipulation techniques commonly used in computer architecture and systems programming.

The programs focus on fundamental operations at the bit level, including working with binary numbers, performing arithmetic operations, and modifying specific bits using efficient algorithms.

### Implemented Programs

- **Reset Rightmost Bit**
  - Demonstrates how to clear the rightmost set bit in a binary number using bitwise operations.

- **16-Bit Addition**
  - Implements addition of two 16-bit numbers while handling carry between lower and higher bytes.

- **Two’s Complement**
  - Calculates the two’s complement of a binary number, which is commonly used for representing negative numbers in computer systems.

### Concepts Demonstrated

- Bitwise operations
- Binary arithmetic
- Two’s complement representation
- Carry handling in multi-byte addition
- Low-level algorithm design

### Purpose

The goal of this project is to practice and understand how computers perform arithmetic and logical operations at the binary level. These concepts are fundamental in areas such as computer architecture, embedded systems, and systems programming.
